# kit-add

## Add integrations to a SvelteKit project

```bash
npx kit-add
```

Open to contributions.

## Built with

- [natemoo-re/clack](https://github.com/natemoo-re/clack/tree/main/packages/prompts)
- [sindresorhus/execa](https://github.com/sindresorhus/execa)
